# my_webdrivers

A Python library for custom logging setup.

## Installation

pip install git+https://github.com/edire/my_logging.git

## Usage

```python
import my_logging

logger = my_logging.NewLogger(__file__, use_cd=True)
logger.info('First Log')
```
