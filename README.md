# Description

A Python library by Dire Analytics for custom logging.

## Installation

pip install git+https://github.com/edire/dlogging.git

## Usage

```python
import dlogging

logger = dlogging.NewLogger(__file__, use_cd=True)
logger.info('First Log')
```
